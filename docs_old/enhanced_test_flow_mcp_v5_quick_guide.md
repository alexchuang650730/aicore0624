## 📝 **使用方法**

### **方法1: Python API (推薦)**
```python
from enhanced_test_flow_mcp_v5 import run_enhanced_test_flow_v5

# 描述你的問題
problem = "VSIX文件安裝失敗，錯誤信息顯示暫不支持安裝插件的3.0.0版本"

# 運行系統
result = await run_enhanced_test_flow_v5(
    requirement=problem,
    mode="developer"
)

# 查看結果
print(result)
```

### **方法2: 直接運行**
```bash

ls -la *report*
```

### **方法3: 完整測試**
